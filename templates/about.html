About Vermicelli

Vermicelli is a version control automation system designed to help you track and manage the versions of your applications efficiently. It implements versioning based on the widely recognized Semantic Versioning (SemVer) principles.

Semantic Versioning Explained

Semantic Versioning (SemVer) is a formal convention for specifying the compatibility of new software versions. According to SemVer, version numbers follow a specific pattern: 

MAJOR.MINOR.PATCH

1. MAJOR version: Incremented when you make incompatible changes.
2. MINOR version: Incremented when you add functionality in a backwards-compatible manner.
3. PATCH version: Incremented when you make backwards-compatible bug fixes.

Semantic Versioning helps ensure that both developers and users have clear expectations about the level of changes introduced with each new release.

How Vermicelli Uses Semantic Versioning

Vermicelli allows you to manage the MAJOR, MINOR, and PATCH versions of your applications through an intuitive web UI or via API requests. It helps you keep track of every change, ensuring smooth and transparent version control.
